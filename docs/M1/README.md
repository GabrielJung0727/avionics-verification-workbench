# M1 — Foundation

M1 산출물 모음. 목표: **시스템 경계, 요구사항 트리, ICD v0, 인증/HF 매핑, 2주 작업 계획**까지 합의.

## 폴더
- `architecture/` — 시스템 경계, 블록 다이어그램, 데이터 플로우
- `requirements/` — System / HLR / LLR 트리 (마크다운 + CSV 시드)
- `icd/` — Interface Control Document v0
- `certification/` — DO-178C / ARP4754A / DO-297 매핑 메모
- `human-factors/` — AC 20-175 / flight deck HF 매핑
- `plan/` — M1 2주 작업 분해 및 Definition of Done

## Exit Criteria
- [ ] 4개 모듈(FCC / Engine I/F / Display / Data Concentrator) 경계 확정
- [ ] 시스템 요구사항 30+ / HLR 50+ 초안
- [ ] ICD v0에 모든 인터-모듈 메시지 등록
- [ ] criticality / DAL 가정 문서화
- [ ] DO-178C objective 매핑 표 v0
- [ ] M2 진입 조건 충족
