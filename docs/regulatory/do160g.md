# DO-160G — Environmental Conditions and Test Procedures for Airborne Equipment

## Scope (1 line)
Standardised laboratory test procedures and environmental conditions
(temperature, vibration, EMI, lightning, altitude, …) for airborne
equipment.

## Status in this repo: **OUT OF SCOPE — explicit declaration**

This is a **simulation-only** workbench. There is no airborne hardware,
no environmental test chamber, no thermocouple, no shake table, no
altitude cabin. **No claim of DO-160G compliance is made or implied.**

| Section | Why it doesn't apply here |
|---|---|
| Temperature & altitude | No physical hardware |
| Vibration & shock | No physical hardware |
| EMI / RF emission | No emitter |
| Lightning indirect / direct | N/A |
| Power input | Software-only |
| Magnetic effect | N/A |

## What this repo does NOT replace
- Any actual environmental qualification activity
- Any inspection that a real airborne unit would undergo
- Any DER / EASA conformance signoff

- Add a `docs/PhaseG/` (hypothetical) for environmental qualification

- Use a HIL-lite physical board first (currently `tools/intelligence/runtime`
  is software loopback)
- Coordinate with a DO-160G test lab — this repo does not attempt to
