# ARP4761A — Aeronautical Safety Assessment

## Scope (1 line)
Methods for conducting the **safety assessment** that drives DAL/IDAL
allocation: FHA, PSSA, SSA, plus supporting analyses (FTA, FMEA, CCA).

## What it covers
- Functional Hazard Assessment (FHA) at aircraft and system levels
- Preliminary System Safety Assessment (PSSA)
- System Safety Assessment (SSA)
- Common Cause Analysis (CCA)
- Fault Tree Analysis (FTA), FMEA / FMES, dependency analysis

## What is intentionally out of scope
- A real FHA / PSSA / SSA — FC-01..12 are illustrative
- Quantitative reliability targets (10^-9 / hour etc.)
- Particular-risk and zonal safety analyses
- A real FTA model with cut sets
