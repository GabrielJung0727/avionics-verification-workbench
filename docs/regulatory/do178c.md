# DO-178C — Software Considerations in Airborne Systems and Equipment Certification

## Scope (1 line)
**The** standard for civil airborne software development & verification
across DAL A..E.

## What it covers
- Software life-cycle data (Plans, Standards, SDD, requirements, code, …)
- Software development processes (planning, requirements, design, coding, integration)
- Software verification (review, analysis, test) including structural coverage
- Configuration management
- Quality assurance
- Certification liaison
- Tool qualification (cross-references DO-330)

## What is intentionally out of scope
- A real Plan for Software Aspects of Certification (PSAC)
- A real Software Configuration Index (SCI) / Software Accomplishment Summary (SAS)
- Substantiated DAL allocation
- Independence claims between activities (single engineer)
