- "System-of-systems verification workbench, not a flight control demo"
- "DO-178C / ARP4754A / DO-297 awareness with traceable evidence"

## 이력서 bullet (초안)
- Designed and built an IMA-style avionics integration & verification workbench: deterministic partition scheduler, ARINC 429-lite + AFDX-lite buses, FCC/Engine/Display surrogates, end-to-end traceability and evidence bundles aligned with DO-178C objectives.
- Implemented requirement-based test runner with structural and MC/DC coverage, fault injection campaigns, and regression dashboard; reproducible evidence bundles re-run from a single archive.
- Integrated HIL-lite bridge (MCU loopback) and human factors evaluation against AC 20-175 alerting and color-usage guidance; documented DAL assumptions, change impact, and tool qualification considerations.
