# M6 — Evidence Bundle, Demo, Portfolio

목표: 모든 산출물을 하나의 evidence bundle로 묶고, 5분 데모와 포트폴리오 자료를 정리해 채용 맥락에서 바로 쓸 수 있게 한다.

## 폴더
- `evidence/` — bundle 포맷, 매니페스트, 재실행 스크립트
- `demo/` — 통합 시나리오, 영상 스크립트
- `portfolio/` — README(영문), 아키텍처 PDF, 이력서 bullet, 포지션별 강조점
- `plan/` — 작업 계획, DoD

## Exit Criteria
- [ ] evidence bundle exporter 동작 (zip + manifest + hash)
- [ ] bundle만으로 동일 결과 재실행 검증
- [ ] DO-178C objective 매핑 표 v1
- [ ] 통합 데모 시나리오 1개 + 5분 영상
- [ ] 영문 README + 아키텍처 다이어그램 PDF
- [ ] 이력서 bullet 3종 (FBW / Display / Verification)
- [ ] LinkedIn 포스트 초안
- [ ] Airbus / Boeing 포지션별 강조 매핑
